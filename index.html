Web Backend (How It Works)
WebSocket & Blynk Gateways

Establishes a persistent, bi-directional socket link directly to your local ESP32 (port 81) for sub-10ms telemetry updates, paired with Blynk Cloud integration for secondary mobile control and remote synchronization.

Smart Auto-Discovery

Performs network scans tracking flura.local, dynamically linking hardware nodes to the web control room without manual IP routing.

Vercel Edge & Local Cache

Hosted on Vercel's high-speed global edge network for instant load times, utilizing local web storage to persist connection states, customized thresholds, and logs.
